01 · Identity concept

The concept

Why this name, and what it means for every design decision.

نغم
Na8am — "melody" in Arabic

Design direction

Na8am carries lightness, rhythm, and personal expression. The template reflects this: soft blues, clean whites, and an airy layout that feels like a portfolio with a voice — not just a résumé. It pairs universal clean design with a subtle Arabic-world personality.

Target user

Developers, designers, and freelancers who want a portfolio that looks professional without looking like a theme. People who care about their online presence but don't want to spend a week building it.

Visual personality

Airy. Trustworthy. Modern without being cold. The sky blue accent creates energy without aggression. The ink-on-white type hierarchy is confident and readable at every viewport.

08 · Voice & copy

Copy tone for Na8am

The template should feel like a real person talking — not a job application. Confident, direct, specific.

Write like this
Use the user's actual name as the hero display title
Short, punchy bio — one to two sentences max in the hero
Skills as plain tags — no percentage bars
Project titles that describe what the project does
"Get in touch" — action verb CTA, not "Contact me"
Social links without labels — the icon is enough
Avoid this
"Welcome to my portfolio" as default hero text — it's filler
Skill progress bars with made-up percentages
"Passionate developer who loves building things"
Project descriptions longer than two lines of body text
All-caps section headers — sentence case only
Generic placeholder text left in by the user

09 · Open issues

Fixes before launch

Seven design and technical issues observed on the live template. Ordered by visual impact.

01
Text breaking mid-word in bio
The About section bio breaks words mid-character — "scal able", "ful l-stack", "user-f riendly". Add overflow-wrap: break-word; word-break: normal; to the bio paragraph container.
High
02
Contact section pink/red background bleed
A color is leaking from below the contact section into the navbar area. Check section background declarations, z-index stacking, and any gradient or color overflow from child elements.
High
03
Inconsistent pill dot colors
The "Professional Journey" section badge renders with a green dot, while all other section badges use Sky Blue. All pill dots must be #3A7BFF — standardize across every section badge component.
High
04
ZoneFolio badge missing
The free tier badge "Made with ZoneFolio" is absent from the template. Add as a fixed-position element: bottom 20px, right 20px, using the dark pill badge style. Link to zonefolio.app.
High
05
Profile image card empty state too tall
When no profile image is uploaded, the card takes up excessive vertical space and the "Profile Image" label looks like a broken component. Reduce card height in the empty state or add a proper designed placeholder with a prompt to upload.
Med
06
Navbar shows "Portfolio" as static text
The left nav wordmark is hardcoded as "Portfolio" — it should pull from the user's site title or display name field dynamically. Update the navbar component to bind to the user data API response.
Med
07
Animation issues on load and scroll
Entrance animations need review — some appear janky or repeat unexpectedly on scroll. Audit all framer-motion or CSS animation triggers. Entrance animations should fire once on mount only, not on scroll re-entry.
Low
